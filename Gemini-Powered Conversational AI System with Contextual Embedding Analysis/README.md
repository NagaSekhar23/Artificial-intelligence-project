### Gemini-Powered Conversational AI System with Contextual Embedding Analysis

A comprehensive project demonstrating advanced chatbot development and natural language processing using the Google Gemini API. This system combines conversational AI with sentence embedding analysis to showcase modern language model capabilities.

#### Features

- **Intelligent Chatbot**: Terminal-based chatbot powered by the Google Gemini API with continuous conversation summarization
- **Context Preservation**: Automatic conversation summarization to maintain context across interactions
- **Sentence Embeddings**: Generation and analysis of semantic embeddings for sentence similarity comparison
- **Rich Responses**: Detailed and flexible responses leveraging Gemini's advanced dialogue handling

#### Project Components

##### Problem 4: Gemini Chatbot
- Interactive terminal chatbot using the Gemini API
- Real-time conversation summarization for context management
- Sophisticated dialogue handling and explanatory capabilities

##### Problem 5: Sentence Embeddings
- Semantic embedding extraction using Gemini's embedding model
- Analysis of sentence similarity and semantic relationships
- Support for multiple sentence inputs with vector representation
